Project Description

This project is a simple Automated File Organizer developed using Python. It categorizes files into different groups such as images, documents, and audio files based on their file extensions.

Features

- Organizes image files
- Organizes document files
- Organizes audio files
- Simple and easy-to-use Python program

Technologies Used

- Python

Project Scope

This project demonstrates the use of Python lists, loops, conditional statements, and file extension handling to organize files automatically.

Output

The program displays categorized files under Images, Documents, and Audio Files.
